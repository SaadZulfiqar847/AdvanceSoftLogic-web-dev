# ProductX — Landing Page

A fully responsive landing page built with pure HTML and CSS (no frameworks), created as part of Week 1 of the Advance Soft Logics Web Development Internship.

## 📋 Project Overview

This project demonstrates core front-end fundamentals:
- Semantic HTML5 structure
- Layout using Flexbox and CSS Grid
- Responsive design with media queries (mobile, tablet, desktop)

## 🛠️ Built With

- HTML5
- CSS3 (no frameworks or libraries)

## 📁 Project Structure
week-1-html-css/
├── index.html
├── style.css
└── README.md

## 🚀 Setup & Usage

1. Clone this repository: git clone https://github.com/SaadZulfiqar847/AdvanceSoftLogic-web-dev.git

2. Navigate into the project folder:
cd AdvanceSoftLogic-web-dev/week-1-html-css
3. Open `index.html` directly in your browser, **or** use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VS Code extension for auto-reloading during development.

No build steps, installations, or dependencies required — this is a static HTML/CSS project.

## ✨ Features

- **Navigation bar** — responsive, collapses to a stacked layout on smaller screens
- **Hero section** — centered heading, tagline, and call-to-action button
- **Features section** — 3-column CSS Grid layout, collapses to single column on mobile
- **Testimonials section** — Flexbox layout with semantic `<blockquote>` elements
- **Contact form** — styled form with name, email, and message fields
- **Footer** — simple copyright footer

## 📱 Responsive Breakpoints

- **Desktop:** default styles (no media query)
- **Tablet (≤768px):** navbar stacks vertically, features grid becomes single-column, testimonials stack
- **Mobile (≤480px):** further reduced font sizes and padding for smaller screens
